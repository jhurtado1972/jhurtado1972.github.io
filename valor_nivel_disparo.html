Valor del nivel de disparo

A continuación, se le mostrará el panel frontal de un osciloscopio analógico.

Usted debe dar la respuesta a la pregunta dada con respecto al nivel de disparo a partir de la señal observada en la pantalla de este osciloscopio. Una vez considere que ya tiene la respuesta, debe escribir el valor medido o calculado en el recuadro (en voltios) y posteriormente podrá verificar si este valor es correcto, seleccionando la opción VERIFICAR.

Recuerde que, para separar las fracciones decimales, debe usar la coma ”,” y NO el punto “.”. Los controles del osciloscopio resaltados en color azul NO se pueden manipular en este ejercicio, pero si se deben tener presentes para la correcta visualización de las señales en la pantalla.

Este ejercicio puede realizarse tantas veces como se desee. Así mismo, se considera como componente de repaso y herramienta de auto verificación del aprendizaje, por lo tanto, no se genera ningún tipo de puntuación ni de calificación.

¿Cuál sería el valor del nivel de disparo (en voltios), para que se pudiera observar la señal sobre la pantalla del osciloscopio tal y como se pidió en el punto anterior (coseno negativo)?

Poner acá la respuesta: V .
