Escala horizontal y medición de periodo

A continuación, se le mostrará el panel frontal de un osciloscopio analógico.

Usted debe dar la respuesta a la pregunta dada con respecto escala horizontal, retícula y medición de periodo a partir de la señal observada en la pantalla de este osciloscopio. Una vez considere que ya tiene la respuesta, debe escribir el valor medido o calculado en el recuadro (en milisegundos -ms-) y posteriormente podrá verificar si este valor es correcto, seleccionando la opción VERIFICAR.

Recuerde que, para separar las fracciones decimales, debe usar la coma ”,” y NO el punto “.”.

Los controles del osciloscopio resaltados en color azul NO se pueden manipular en este ejercicio, pero si se deben tener presentes para la correcta visualización de las señales en la pantalla.

Este ejercicio puede realizarse tantas veces como se desee. Así mismo, se considera como componente de repaso y herramienta de auto verificación del aprendizaje, por lo tanto, no se genera ningún tipo de puntuación ni de calificación.

Indicar el periodo de la señal que se observa en pantalla, a partir de las mediciones sobre la retícula y la escala horizontal seleccionada en el punto anterior.

Poner acá la respuesta: μs.
