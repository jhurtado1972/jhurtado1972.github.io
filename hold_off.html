Hold-Off

A continuación, se le mostrará el panel frontal de un osciloscopio analógico.

Usted podrá mover y seleccionar la posición del control de hold-off (resaltado en color amarillo) de este osciloscopio. Una vez usted considere que ha puesto el control en la posición adecuada de acuerdo con la pregunta dada, podrá verificar si esta es la posición correcta, seleccionando la opción VERIFICAR.

Los controles del osciloscopio resaltados en color azul NO se pueden manipular en este ejercicio, pero si se deben tener presentes para la correcta visualización de las señales en la pantalla.

Este ejercicio puede realizarse tantas veces como se desee. Así mismo, se considera como componente de repaso y herramienta de auto verificación del aprendizaje, por lo tanto, no se genera ningún tipo de puntuación ni de calificación.

El osciloscopio tiene como entrada una señal que está modulada en amplitud. Esta señal es de difícil visualización estable en la pantalla. Para estabilizar la visualización de esta señal, usted podrá manipular el control de hold-off, el cual permite rechazar algunos pulsos de disparo, de tal forma que los pulsos de compuerta coincidan en el mismo punto de inicio de la señal para que se dé una visualización estable en la pantalla.
